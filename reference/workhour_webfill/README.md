# 科技管理平台工时填报自动化解决方案 🚀

## 📖 项目简介

本项目为科技管理平台的工时填报提供自动化解决方案，帮助用户快速完成繁琐的工时填写工作，显著提高工作效率。

## 🎯 解决方案概览

提供两种不同的自动化方案，您可以根据自己的技术水平和需求选择：

| 方案 | 技术栈 | 难度 | 推荐指数 | 适用人群 |
|------|--------|------|----------|----------|
| **方案一：浏览器脚本** | JavaScript/Tampermonkey | ⭐⭐ | ⭐⭐⭐⭐⭐ | 所有用户 |
| **方案二：Python自动化** | Python/Selenium | ⭐⭐⭐ | ⭐⭐⭐⭐ | 有编程基础的用户 |

---

## 🌟 方案一：浏览器用户脚本（推荐）

### 💡 特点
- **简单易用**：一键安装，无需编程基础
- **实时操作**：在网页上直接使用，所见即所得
- **轻量高效**：不占用系统资源
- **安全可靠**：只在指定网站运行，无数据泄露风险

### 📁 相关文件
- `工时自动填写脚本.js` - 主脚本文件
- `使用说明.md` - 详细使用教程

### 🎮 核心功能
✅ 一键自动填写所有必要字段  
✅ 智能日期识别（当前日期）  
✅ 随机工作内容模板  
✅ 可视化设置界面  
✅ 快捷键支持（Ctrl+Shift+F）  
✅ 实时状态提示  

### 🚀 快速开始
1. 安装 [Tampermonkey](https://tampermonkey.net/) 浏览器扩展
2. 复制 `工时自动填写脚本.js` 中的代码并安装
3. 访问工时填报页面，点击右上角的 "🚀 一键填写工时" 按钮

## 🔧 方案二：Python自动化脚本

### 💡 特点
- **功能强大**：可扩展性强，支持复杂的自动化逻辑
- **批量处理**：可以处理多天工时批量填写
- **高度定制**：完全可控的填写流程
- **跨平台**：支持Windows、Mac、Linux

### 📁 相关文件
- `方案二_Python自动化.py` - Python自动化脚本
- `requirements.txt` - Python依赖包列表

### 🎮 核心功能
✅ 全自动浏览器操作  
✅ 智能等待和错误处理  
✅ 详细的日志记录  
✅ 可配置的填写参数  
✅ 支持无头模式运行  
✅ 异常恢复机制  

### 🚀 快速开始
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行脚本
python 方案二_Python自动化.py
```

## 🎛️ 配置说明

### 浏览器脚本配置
```javascript
const CONFIG = {
    workNature: '技术开发',           // 工作性质
    defaultWorkHours: 8,             // 默认工时
    defaultStartTime: '09:00',       // 开始时间
    defaultEndTime: '17:00',         // 结束时间
    defaultRemark: '工作正常完成',    // 默认备注
    autoFillCurrentDate: true        // 自动填充当前日期
};
```

### Python脚本配置
```python
self.config = {
    'default_work_hours': 8,          # 默认工时
    'default_start_time': '09:00',    # 开始时间
    'default_end_time': '17:00',      # 结束时间
    'default_work_nature': '技术开发', # 工作性质
    'wait_timeout': 10,               # 等待超时时间
}
```

## 🔧 故障排除

### 常见问题

#### 问题1：脚本不运行
**原因**：页面结构变化或网络问题  
**解决**：
1. 刷新页面重试
2. 检查浏览器控制台错误信息
3. 确认脚本是否已启用

#### 问题2：填写不完整
**原因**：页面加载较慢或元素定位失败  
**解决**：
1. 等待页面完全加载后再操作
2. 多次点击填写按钮
3. 手动补充遗漏字段

#### 问题3：Python脚本运行错误
**原因**：环境配置或依赖问题  
**解决**：
```bash
# 重新安装依赖
pip install -r requirements.txt --upgrade

# 检查Chrome版本
google-chrome --version
```
