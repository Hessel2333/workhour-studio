# AutoFill WorkHour (工时表生成器)

一个简单易用的工时统计表生成器，帮助您快速生成标准格式的工时统计Excel表格。

## 功能特点

- 📅 支持年月选择，自动匹配日期与星期
- 📊 生成半小时为单位的时间段（8:00-17:00）
- 🍽️ 自动跳过午休时间（11:30-13:00）
- 📋 支持工作模板配置
- 📌 支持固定时间安排
- 💾 支持导入导出配置
- 🌓 支持深色模式
- 📱 响应式设计

## 技术栈

- Next.js 14
- React 18
- TailwindCSS
- TypeScript
- XLSX

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm run start
```

## 部署

本项目已部署在 Vercel 平台上，访问地址：[AutoFill WorkHour](https://autofill-workhour.vercel.app)

## 使用说明

1. 基本信息设置
   - 选择年份和月份
   - 输入员工姓名

2. 工作模板配置
   - 添加常用的工作内容模板
   - 设置工作性质、类别等信息
   - 配置随机出现的权重

3. 固定时间安排
   - 设置固定的工作安排
   - 指定具体的时间段和工作内容

4. 生成工时表
   - 点击"生成工时统计表"按钮
   - 自动下载Excel格式的工时统计表

## 配置导入导出

- 导出配置：将当前的工作模板和固定安排导出为JSON文件
- 导入配置：从之前导出的JSON文件中恢复配置
