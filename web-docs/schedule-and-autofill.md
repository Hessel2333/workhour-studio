# 日程与自动补全

日程是维护工时的主入口。每个时间块都对应一条工时记录，可以拖拽移动、拉伸调整长度，也可以点击打开编辑面板。

## 周视图

周视图左侧显示常用项目和工作项，右侧显示一周时间轴。卡片颜色跟随项目，卡片内显示关联项目或备注、工作性质、工作形式、时间范围和工时长度。

## 编辑时间块

点击时间块可以编辑日期、时间、工作性质、工作类别、关联项目、工作形式、备注和共同完成人。工作类别、项目和形式会按菜单规则自动过滤。

## 自动补全

自动补全会识别工作日内的空白时间，再从启用模板中挑选内容生成记录。补全后可以直接在日程里拖拽、拉伸或点击编辑。

单独卡片不会生成 `0.5h` 工时，优先使用 `1h`、`1.5h`、`2h` 这类更自然的时间块。

## 复制与撤销

`复制昨日` 会把前一天的记录复制到当前日期。`撤销` 会恢复上一次日程操作前的记录状态，适合快速试错。
