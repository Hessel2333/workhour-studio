# 快速开始

workhour-studio 用来维护本地工时数据。项目、模板、日程和工时记录都保存在本地数据库里，Excel 只用于导入、导出和兼容旧填报流程。

## 推荐上手顺序

1. 先在 `导入导出` 中导入当前维护的 Excel。
2. 到 `项目库` 检查项目类别，必要时调整项目名称、项目号和类别。
3. 到 `模板库` 检查常用工作内容，保留真正会重复使用的模板。
4. 在 `日程` 中维护每天的时间块，拖拽移动，拉伸调整时长。
5. 到 `工时表` 检查正式记录，再按月导出 Excel。

## 数据来源

本地数据库是唯一数据源。导入 Excel 后，系统会读取项目清单、一级二级菜单和 2026 年及以后的填报记录；历史填报也会沉淀为可编辑模板。

## 工作台分工

- `日程` 负责安排和编辑每天的时间块。
- `项目库` 负责维护可选项目。
- `模板库` 负责维护可复用的工作内容。
- `工时表` 负责查看和补录正式工时。
- `分析` 负责查看范围内的工时统计。
- `导入导出` 负责 Excel 和旧配置文件的进出。
