# 导入导出

Excel 是兼容格式，不是长期维护数据源。日常维护以本地数据库为准，需要填报平台时再导出月度 Excel。

## 导入 Excel

导入时会读取 `项目清单`、`一级和二级菜单`，并读取 2026 年及以后的月度记录。`Sheet2` 里的旧模板不会导入。旧列名 `内容属性` 会映射为 `关联项目`。

再次导入 Excel 时，系统会替换 Excel 来源的记录、项目和模板。已经手动编辑过的项目或模板会作为本地维护内容保留。

## 导入 JSON

旧生成器的 JSON 配置会转换为模板。随机时段、固定安排和周末讲堂会保留对应类型，便于继续用于自动补全。

## 导出 Excel

导出按当前月份生成兼容旧 Tampermonkey 脚本的 Excel。字段顺序保留平台需要的工作日期、开始时间、结束时间、工作性质、工作类别、关联项目、工作形式、备注和共同完成人。

## 历史记录

每次导入导出都会保留历史摘要。失败时会记录错误信息，方便回头检查文件或数据格式。
